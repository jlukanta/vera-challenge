[USAGE]

./lookup relative_path_to_dump_file address1 address2 ...
eg. ./lookup symbolDumps/symbol_dump2.txt 0x2d3c8cf9 0x3c000

---------------------

[SETUP]

On command line / terminal, type make
[*] Executable will be located on the same directory as this README file as ./lookup
[*] Compiler: gcc
    Configured with: --prefix=/Applications/Xcode.app/Contents/Developer/usr --with-gxx-include-dir=/usr/include/c++/4.2.1
    Apple LLVM version 6.1.0 (clang-602.0.53) (based on LLVM 3.6.0svn)
    Target: x86_64-apple-darwin14.4.0
    Thread model: posix

---------------------

[TESTS]

To run test cases
1. Open XCode (mine is XCode 6.4)
2. Select LookupSymbolTests as build target
3. Select Product > Test.
