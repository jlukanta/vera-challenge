[CHALLENGE]

Multiplayer Snake

Write a massively multiplayer version of the game snake. Your game should let a large number of players join over the Internet and play against each other.

—————————————————


[RUNNING PROJECT]
1. Install node
2. Run npm install
3. node server.js
4. Open your browser http://localhost:3000
A board can handle at most 10 players before a new board is created.
